# v19 여행 모드 고도화

## 목적

여행 탭을 단순 장소/액션 목록에서 실제 여행 준비용 화면으로 확장합니다.

## 추가된 구조

```text
TravelChecklistItem
TravelReservationItem
TravelMemoItem
```

## 주요 기능

### 1. 여행 체크리스트

출발 전 또는 Day별 준비물을 관리합니다.

예:

```text
여권/신분증
환전/카드
보조배터리
유심/eSIM
상비약
숙소 바우처
```

### 2. 예약/티켓

예약번호, 시간, 장소, 티켓 정보를 저장합니다.

예:

```text
호텔 예약
항공권
USJ 입장권
렌터카 예약
식당 예약
```

### 3. 오프라인 여행 메모

인터넷이 안 되는 상황에서도 볼 수 있는 메모 구조입니다.

예:

```text
호텔 주소
현지 교통 팁
주의사항
아이 준비물
식당 후보
```

### 4. Day별 타임라인

여행 액션, 예약/티켓, 메모를 Day별로 묶어 보여줍니다.
